DeepZoomContainer, improved MultiScaleImage in Silverlight

Written by Won Dong(fkiller@gmail.com), ROH Holding Company

1. Project Description

DeepZoom enables powerful functions to build large-scale image-based projects such as GIS application, games, and more. DeepZoomContainer is a package of classes that expands DeepZoom functionality.

2. Project/Directory Description

/DeepZoomContainer
Main class package project that has several useful classes, such as PushPin, WheelMouseListener, and DeepZoom Container.

/DeepZoomContainer.Test
Silverlight application that test DeepZoomContainer class.

/DeepZoomContainer.Test.Web
Web application that contains DeepZoomContainer.Test silverlight application.

/PathAnimation
Silverlight Path Animation Class

/Document
XML Documentation of classes
